# Contributing to Soroban Explorer

Thank you for your interest in contributing to Soroban Explorer! This document provides guidelines and information about contributing to this project.

### Reporting Bugs

If you find a bug, please create an issue with:
- A clear, descriptive title
- Detailed steps to reproduce the issue
- Expected behavior vs actual behavior
- Screenshots if applicable
- Your environment (OS, browser, Node version)

### Suggesting Features

We love new ideas! To suggest a feature:
- Check if it's already been suggested in issues
- Create a new issue with the `enhancement` label
- Describe the feature and its benefits
- Provide examples or mockups if possible

### Pull Requests

1. **Fork the repository** and create your branch from `main`
2. **Make your changes** following our coding standards
3. **Test your changes** thoroughly
4. **Update documentation** if needed
5. **Submit a pull request** with a clear description
