# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-15

### Added
- Initial release of Soroban Explorer
- Contract Explorer feature with search functionality
- Network Statistics dashboard
- Recent Contracts browser with filtering
- Multi-network support (Testnet/Mainnet)
- Responsive design with dark theme
- Integration with Stellar SDK
- External links to Stellar Expert
- Copy-to-clipboard functionality
- Real-time network health monitoring

### Features
- **Contract Explorer**
  - Search contracts by ID
  - View contract details and metadata
  - Display available contract functions
  - Network selection (Testnet/Mainnet)

- **Network Statistics**
  - Total contracts counter
  - Transaction volume tracking
  - Active contracts metrics
  - Recent activity feed
  - Network health indicators

- **Recent Contracts**
  - List of recently deployed contracts
  - Filter by contract type
  - Filter by network
  - Contract call statistics
  - Deployer information

### Technical
- React 18.2 with TypeScript
- Vite build system
- Tailwind CSS for styling
- Lucide React for icons
- Stellar SDK integration

### Documentation
- Comprehensive README
- Contributing guidelines
- MIT License
- Project structure documentation
